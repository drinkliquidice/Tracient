This is the github repo for [Tracient](https://tracient.pages.dev/login), a service which allows organizations to keep track of their member and assets.

Built with Kumon centers in mind, you are able to keep log of the sign-in and sign-out hours of members, or students in this case alongside assets such as books
which can we checked out by these members.

More updates are the come as this project is still in the dev stages.

STACK:
Frontend: 
- SolidJS for reactivity
- TailwindCSS for design

Backend:
- MongoDB for storage
- Beanie for document formatting
- Twilio for notifications

Hosted on Cloudflare Pages and Render
